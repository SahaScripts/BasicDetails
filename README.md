# Student Details Web Page

## Description

This project is a **simple HTML-based Student Details Page** designed to display key academic information using basic inline styling for clarity.
General student details are highlighted in **green**, while course-related details are shown in **blue** for easy visual separation.
It serves as a beginner-friendly practice project for understanding HTML structure and inline styling.

---

## Features

* Displays essential student details:

  * Name
  * Registration Number
  * Year of Joining
  * Batch
  * School
  * Course
  * Course Code
* Clean and minimal HTML layout
* Color-coded text for better readability
* Works on all modern web browsers
* No dependencies or external libraries required

---

## Requirements

* Any modern web browser (Chrome, Edge, Firefox, Safari)

---


## Usage

Open the `index.html` file in your browser to view the student details.
You may edit the values or styling directly inside the HTML file if customization is needed.
